Movement & Physics

move_speed 1.0
Movement speed multiplier
Values: 0.1-5.0 (Engine), >5.0 (Custom velocity)
super_jump 1
Supa jump
0 = Off, 1 = On
jump_height 300
Jump upward force
Default: 300 | Examples: 500, 750, 1000
jump_multi 1
Jump momentum multiplier
Examples: 1.0 (default), 1.5, 2.0
jump_mode 1
Jump cooldown type
0 = Time-based, 1 = Ground-based (default)
jump_cooldown 1
Jump cooldown in seconds( when using "jump_mode 0" for jumps instead of the default "super_jump" method)
Examples: 0.3 (fast), 1.0 (default), 2.0 (slow)
low_grav 1
Low gravity mode
0 = Off, 1 = On
low_grav_strength 25
Gravity reduction strength
Default: 25 | Examples: 20 (light), 50 (strong)
scaletime 1.0
Game timescale multiplier
Range: 0.1-20.0 | Examples: 0.5 (slow-mo), 1.0 (normal), 2.0 (fast)
nc_enabled 1
Toggle noclip with Jump+Melee buttons
0 = Off, 1 = On (default)
nc_slow 8.0
Noclip slow speed
Default: 8.0
nc_normal 25.0
Noclip normal speed (Sprint)
Default: 25.0
nc_fast 100.0
Noclip fast speed (Sprint+Tactical)
Default: 100.0
Notice: When setting "move_speed" above 5 its start using a script based speed system, you may just start flying at times..

Examples: iw8_ar_mike4 (M4A1), iw8_sn_alpha50 (AX-50), s4_ar_stango44 (STG-44)

-1 = Random camo, 0-999 = Specific variant

Maximum weapons to carry

Auto-switch to given weapon

Notes & Setup

Download & Installation

Step 1: Download the script file from Discord
If you have issues, dm / tag me
Download from #community-extras
Step 2: Place script.gscbin in your game directory
The file should be in the same folder as ModernWarfare.exe
Step 3: Launch the game and join a private match
The script will load automatically at match connection
Notice: May crash / not load at all on some builds.

General Tips

  • Player listing shows the ID (#0, #1, #2, etc.) that identifies each player
  • Bans persist until map restart
  • Some omnvars may cause crashes or unexpected stuff
  • Noclip can sometimes toggle when you don't want it to, especially when using melee weapons
  • Infinite ammo has issues with akimbo weapons in some of the latest builds
  • Barrier removal may break modes such as snd

Finding Resources

  • Sound Aliases: Search dumps for sounds used with playlocalsound() and similar
  • Scriptables / Omnvars: Search dumps for setscriptablepartstate() and setclientomnvar() calls
  • Splash Messages: Search dumps or CSV files
  • VFX Paths: Full paths start with vfx/
